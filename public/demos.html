如何使用演示

1️⃣

选择演示

从上面的列表中选择一个您感兴趣的演示

2️⃣

阅读说明

每个演示都有详细的说明和代码示例

3️⃣

执行操作

使用提供的表单和按钮执行实际操作

4️⃣

查看结果

实时查看操作结果和返回的数据

⚠️ 重要提示
  • 所有演示都使用 Stripe 的测试环境，不会处理真实资金
  • 您需要有效的 Stripe API 密钥才能执行某些操作
  • 演示中的 API 调用需要后端服务器支持
  • 建议先阅读相关教程，再使用演示进行实践
  • 测试卡号仅用于测试，不能用于真实支付
